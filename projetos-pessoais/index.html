PortfolioHUB

Gerenciamento e Exibição de Projetos Acadêmicos

Sobre

O PortfolioHUB é uma plataforma desenvolvida para organizar, armazenar e exibir projetos acadêmicos e pessoais.

Funcionalidades

  • Gerenciamento de Projetos
  • Integração com GitHub
  • Controle de Versão
  • Compartilhamento de Código
  • Gestão de Usuários

Projeto Destaque

Loja de Material de Construção em Python

Sistema desenvolvido para gerenciamento de estoque e vendas.

Tecnologias Utilizadas

HTML, CSS, Git, GitHub e Google Gemini.
